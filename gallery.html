Two things this page cannot show 1. index.css contains @tailwind directives that only resolve through the PostCSS build. Loaded raw here, no Tailwind utility class works — so RM Pro Forma specimens (which are built from utilities) render structurally but unstyled. The Forecast and ace-* specimens are hand-written CSS and render exactly as in the app. 2. Fonts load from Google Fonts via @import; offline, everything falls back to system stacks.
